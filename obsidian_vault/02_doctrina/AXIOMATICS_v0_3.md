# AXIOMATICS OF THE HEAVENS v0.3 (2025-12-30)

Supersedido por: [[AXIOMATICS_v0_4]]

## 1. Ontological Axioms

1.1. The Heavens are a continuous, ordered, and differentiable manifold $\mathcal{H}$, embedded in four-dimensional spacetime $\mathbb{R}^4$.

1.2. Celestial bodies are persistent, locally differentiable trajectories within $\mathcal{H}$, each with unique worldlines $\gamma_i(t)$.

1.3. The observer is a worldline $\gamma_O(t)$ with a privileged local frame.

---

## 2. Epistemic Axioms

2.1. All astrological knowledge is conditional on the observer's spacetime coordinates $(t_O, x_O, y_O, z_O)$.

2.2. Astrological statements are formalized as mappings $f: \mathcal{H} \to \mathcal{S}$, where $\mathcal{S}$ is a semantic space of interpretations.

2.3. The truth value of an astrological statement is a function of both celestial configuration and observer context.

---

## 3. Computational Axioms

3.1. The astrological chart is a finite, computable projection $\pi: \mathcal{H} \to \mathbb{R}^n$, parameterized by observer context and epoch.

3.2. All chart calculations are reproducible, given identical initial conditions and ephemeris data.

3.3. The system supports algorithmic extension without loss of generality.

## 6. Relocation Field Axioms *(Added 2026-03-13)*

6.1. The geographic field $\mathcal{F}(\phi, \lambda)$ is a scalar function over the Earth's surface — the Harmony Field evaluated at each relocation coordinate for a fixed natal chart.

6.2. The relocation field is not a prediction of outcomes; it is a measure of structural resonance.

6.3. The natal chart evaluated at birth coordinates defines the reference value $\mathcal{F}_0$. The **relocation delta** $\Delta\mathcal{F}(\phi, \lambda) = \mathcal{F}(\phi, \lambda) - \mathcal{F}_0$ is the operationally meaningful quantity.

7.2. **Domain Opacity Axiom**: The global relocation field $\mathcal{F}(\phi, \lambda)$ is semantically opaque when the consultee's question is domain-specific.

7.4. **Selector Epistemic Axiom**: The domain selector is not a navigation feature — it is the implementation of Axiom 7.2.
